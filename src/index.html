The pages on this site demonstrate the built-in AI capabilities that are now available as developer previews in Microsoft Edge Canary and Dev. These experimental APIs allow sites and extensions to access AI capabilities via a local version of the Azure Phi-4-minni, a powerful small language model that is built into the browser.

To start using the prompt and writing assistance APIs today in Microsoft Edge Canary or Dev, check out the API documentation, and explore the source code of the demo pages on GitHub.

To share any feedback about the APIs, please comment on the feedback GitHub issue.
